# User Badge Component

유저 등급을 나타내는 뱃지입니다.

<Description>White : 초기 등급 (default)</Description>
<Description>Green : 질문 1개 이상, 댓글(답글) 5회 이상</Description>
<Description>Blue : 질문 5개 이상, 댓글(답글) 10회 이상</Description>
<Description>Red : 질문 15개 이상, 댓글(답글) 20회 이상</Description>
<Description>Gold : Gold 질문 30개 이상, 댓글(답글) 50회 이상</Description>
